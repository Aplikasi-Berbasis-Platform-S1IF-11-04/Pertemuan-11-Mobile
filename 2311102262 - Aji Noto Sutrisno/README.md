#  Personal Finance App

Aplikasi manajemen keuangan pribadi berbasis Flutter yang membantu pengguna mencatat pemasukan, pengeluaran, dan memantau kondisi keuangan sehari-hari.

---

##  Fitur

| Fitur | Deskripsi |
|---|---|
|  **Dashboard** | Tampilan utama dengan ringkasan saldo, pemasukan, dan pengeluaran |
|  **Tambah Transaksi** | Form input transaksi dengan kategori dan toggle pemasukan/pengeluaran |
|  **Kategori** | Pengelompokan pengeluaran berdasarkan kategori (Makanan, Transportasi, dll) |
|  **Profil** | Informasi pengguna, target tabungan, dan progres pencapaian |
|  **Notifikasi** | Pengingat dan notifikasi kondisi keuangan |
|  **Pengaturan** | Konfigurasi notifikasi, mode gelap, biometrik, dan mata uang |
|  **Tentang** | Informasi aplikasi dan developer |
